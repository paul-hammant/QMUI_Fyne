# Toast View

**Component:** `QMUIToastView`
**Category:** Feedback

## Description

A lightweight toast notification system for displaying brief messages, loading indicators, and status feedback. Auto-dismisses after configurable duration.

## Features

- Loading, success, error, and info states
- Custom content view support
- Customizable position and animation
- Background blur support
