# Modal Presentation

**Component:** `QMUIModalPresentationViewController`
**Category:** Presentation

## Description

A customizable modal presentation controller that provides flexible presentation styles, animations, and dimming view configurations. Supports presenting any view controller with custom positioning and sizing.

## Features

- Multiple presentation styles (center, bottom, full screen)
- Customizable dimming view with tap-to-dismiss
- Keyboard-aware positioning
- Custom animation support
- Window-level or view-level presentation
