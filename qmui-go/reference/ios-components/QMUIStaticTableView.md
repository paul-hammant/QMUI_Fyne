# Static Table View

**Component:** `QMUIStaticTableView`
**Category:** Tables

## Description

A data-driven static table view for settings screens and form layouts. Supports various cell types including switches, disclosure indicators, and checkmarks.

## Features

- Declarative cell configuration
- Multiple accessory types
- Section headers and footers
