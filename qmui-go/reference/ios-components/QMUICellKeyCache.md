# Cell Height Cache

**Component:** `QMUICellKeyCache`
**Category:** Performance

## Description

Automatic cell height caching for UITableView and UICollectionView. Improves scrolling performance by caching calculated heights.

## Features

- Automatic height caching
- Key-based cache invalidation
- UITableView and UICollectionView support
- Estimated height support
