# Float Layout View

**Component:** `QMUIFloatLayoutView`
**Category:** Layout

## Description

A flow layout view that arranges subviews horizontally and wraps to new lines when needed. Ideal for tags, chips, and dynamic content.

## Features

- Horizontal flow layout
- Automatic line wrapping
- Configurable spacing
